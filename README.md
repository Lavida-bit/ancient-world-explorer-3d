# 🌍 Ancient World Explorer 3D

Ancient World Explorer 3D is an interactive web application built with Three.js that lets users explore famous ancient monuments around the world on a 3D globe.

## Features

- 🌎 Interactive 3D Earth
- 📍 Clickable monument markers
- 🔎 Search for monuments
- 📖 Educational information about each site
- ⚖️ Mainstream historical explanations
- 🛸 Clearly labelled alternative theories (for educational purposes only)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js

## Project Status

🚧 Currently under development.

## Educational Notice

This project is intended for educational purposes. Alternative theories are presented as speculative and are not supported as established historical facts.
